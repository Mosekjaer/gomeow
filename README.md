# gomeow

A Go library for game memory reading, overlay rendering, and GUI creation.

## Overview

gomeow provides a comprehensive toolkit for:
- Process memory reading and writing
- Pattern scanning (AOB/signature scanning)
- Overlay window rendering with raylib
- Immediate-mode GUI components
- Cross-platform support (Windows + Linux)

## Installation

```bash
go get gomeow
```
